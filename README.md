# 🍎 Apple Website Clone (React)

A modern **Apple landing page clone** built using **React.js** and deployed on **Vercel**.

This project recreates Apple's clean product-focused homepage design using reusable React components, responsive layouts, and modern frontend development practices.

---

## 🚀 Live Demo

🔗 **Live Website:**
https://react-apple-livid.vercel.app

---

## 📌 Project Overview

This project replicates the visual layout and user experience of Apple's official homepage.

The goal of this project was to practice:

* Component-based architecture
* Modern React development
* Responsive UI design
* Real-world layout recreation
* Production deployment workflow

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/tabisso/react-apple
cd apple-react-clone
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm start
```

App runs at:

```
http://localhost:3000
```

---

## ☁️ Deployment (Vercel)

This project is deployed using **Vercel**:

```bash
npm run build
```

Then connected directly to GitHub for automatic deployment.

Every push triggers a new production build.
